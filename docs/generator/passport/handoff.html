表紙は PNG 全面配置に対応: Image（BoxFit.cover）を最下層に置き、サイズ調整は Transform.scale(scale: coverImageScale)（中央基準・0.5〜2.0）。帯・環境光・グレインは画像の上に重ねる。
質感（Paper「sinta」の再現） — 数値の正はジェネレーターの .spine レイヤー / Dart 書き出し
  • 帯（クロステープ）: 地色 tapeColor の上にグラデーション3層 — ①外端のエッジ線（黒22% → 白ハイライト → 黒10% の細線）②テープ中央の丸みシーン（白系）③テープ先の折り溝（黒16% → フェード、その先に僅かな照り返し）
  • 環境光: 表紙上部に白10%→透明（26%位置まで）、下部に黒14%→透明の落ち影
  • グレイン: 微細ノイズ opacity 3〜6%（feTurbulence 相当。Flutter ではノイズ画像の Image 重ねか省略可）
  • 閉時のドロップシャドウ: 大きく柔らかい影（blur 34px 相当・下方向）。ズーム完了で消す
  • 開く途中の落ち影: ページの背表紙側に黒28%→透明45% のグラデーション。開き進捗と共に opacity 1→0

PASSPORT チップの状態（Figma 9044-41898）
  • チップはパスポートの開閉トグル: 閉→タップで開く、開いている間→タップで「戻る」と同じ閉じシーケンス
  • 開いている間（選択中）は dark/light を反転: bg rgba(0,0,0,.65)→rgba(255,255,255,.85)、border 白15%→黒15%、文字 白→黒（0.3s ease）。ブックアイコンはそのまま
  • パスポート表示中もヘッダー（チップ・カウンター・ヘルプ）は暗幕の上に残す
Web プロトタイプとの対応（参考）
  • 回転は CSS transform-origin: 0% 50%; rotateY(−176deg) ＋ backface-visibility: hidden の2面。Flutter の rotateY は座標系の解釈で逆に見える場合があるため、ジェネレーターと見比べて「手前に持ち上がって開く」ことを確認し、逆なら符号を反転してください
  • イージングは CSS cubic-bezier(a,b,c,d) = Flutter Cubic(a,b,c,d) でそのまま互換
  • タイミング・色・質感の数値はすべてジェネレーターの「Dart をコピー」出力が正（表紙色などの調整値が焼き込まれます）
