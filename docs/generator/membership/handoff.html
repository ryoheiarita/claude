会員証カード スタイル実装指示

マイページの会員証カード（Figma Design_System_V2 809:255 / Renewal_UI 1812:15163）のスタイル仕様です。固定サイズ 358 × 231・角丸 24。1枚の会員証画像を「背景ぼかし」と「メイン画像」の両方に使うのが要点です。動きの正解・クロップ調整はジェネレーターを参照してください。

プレビュー

実寸 358 × 231。下の見た目がゴールです（画像はサンプル）。

Artist・FCname
Nickname

背景のぼかし（blur 64・不透明度 50%）は、マイページ上部背景にも同じ画像で敷かれます。

レイヤー構成（下 → 上）

カードは ClipRRect(radius: 24) で角丸クリップした Stack。枠線だけは最前面に重ねる（クリップに食われないように）。
  • ① 背景ぼかし — 同一画像を全面 BoxFit.cover → Gaussian blur σ=32。ぼかし縁の透けを防ぐため Transform.scale(1.35) で少し広げる（Figma Frame 71 の backdrop-blur 32 に相当）
  • ② 黒 30% オーバーレイ — Color(0x4D000000) を全面に
  • ③ メイン画像 — 上6 / 左6 / 右6 の位置、高さ 195（= 346 × 195）。角丸は上 20 / 下 12 の非対称。BoxFit.cover
  • ④ テキスト — Artist・FCname（左下）/ Nickname（右下）。左右 16・下 8 の余白
  • ⑤ グラデ枠線 — 内側 1px、#FFFFFF 30% → 10% の水平グラデ（CustomPaint で最前面）

Flutter Widget（コピーして使用）

image に会員証画像を渡すだけ。追加パッケージ不要（dart:ui と material のみ）。

membership_card.dart 
import 'dart:ui' as ui;
import 'package:flutter/material.dart';

/// 会員証カード
/// Figma: Design_System_V2 `809:255` / Renewal_UI `1812:15163`
/// 固定サイズ 358 × 231、角丸 24。
/// [image] の同一画像を「背景ぼかし」と「メイン画像」の両方に使う。
class MembershipCard extends StatelessWidget {
  const MembershipCard({
    super.key,
    required this.image,
    this.artist = 'Artist・FCname',
    this.nickname = 'Nickname',
  });

  final ImageProvider image;
  final String artist;
  final String nickname;

  @override
  Widget build(BuildContext context) {
    const label = TextStyle(
      fontFamily: 'Figtree',
      fontSize: 12,
      height: 1.0,
      letterSpacing: -0.24,
      color: Color(0xB3FFFFFF), // #FFFFFF 70%
    );

    return SizedBox(
      width: 358,
      height: 231,
      child: Stack(
        children: [
          // 本体（角丸 24 でクリップ）
          ClipRRect(
            borderRadius: BorderRadius.circular(24),
            child: Stack(
              fit: StackFit.expand,
              children: [
                // ① 背景ぼかし：同一画像を全面 cover → Gaussian blur 32
                //    Transform.scale で少し広げ、ぼかし縁の透けを防ぐ
                ImageFiltered(
                  imageFilter: ui.ImageFilter.blur(sigmaX: 32, sigmaY: 32),
                  child: Transform.scale(
                    scale: 1.35,
                    child: Image(image: image, fit: BoxFit.cover),
                  ),
                ),
                // Frame 71：黒 30% オーバーレイ
                const ColoredBox(color: Color(0x4D000000)),
                // ② メイン画像：上6 / 左6 / 右6、高さ 195、角丸 上20・下12
                Positioned(
                  left: 6,
                  top: 6,
                  right: 6,
                  height: 195,
                  child: ClipRRect(
                    borderRadius: const BorderRadius.only(
                      topLeft: Radius.circular(20),
                      topRight: Radius.circular(20),
                      bottomLeft: Radius.circular(12),
                      bottomRight: Radius.circular(12),
                    ),
                    child: Image(image: image, fit: BoxFit.cover),
                  ),
                ),
                // ③ テキスト：左下 / 右下（下8・左右16）
                Positioned(left: 16, bottom: 8, child: Text(artist, style: label)),
                Positioned(right: 16, bottom: 8, child: Text(nickname, style: label)),
              ],
            ),
          ),
          // ④ グラデ枠線：内側 1px、#FFFFFF 30% → 10%（水平）
          Positioned.fill(
            child: IgnorePointer(
              child: CustomPaint(painter: _GradientBorderPainter()),
            ),
          ),
        ],
      ),
    );
  }
}

class _GradientBorderPainter extends CustomPainter {
  @override
  void paint(Canvas canvas, Size size) {
    final rect = Offset.zero & size;
    // stroke 中心を 0.5 内側に寄せて内側 1px にする
    final rrect = RRect.fromRectAndRadius(
      rect.deflate(0.5),
      const Radius.circular(23.5),
    );
    final paint = Paint()
      ..style = PaintingStyle.stroke
      ..strokeWidth = 1
      ..shader = const LinearGradient(
        begin: Alignment.centerLeft,
        end: Alignment.centerRight,
        colors: [Color(0x4DFFFFFF), Color(0x1AFFFFFF)], // 30% → 10%
      ).createShader(rect);
    canvas.drawRRect(rrect, paint);
  }

  @override
  bool shouldRepaint(covariant CustomPainter oldDelegate) => false;
}

マイページへの配置

会員証カードはマイページ上部（Figma 1812:15161、左右 16 余白 = 幅 358）に置かれ、同じ会員証画像がページ最上部の背景にも敷かれます。
  • ページ上部背景 — 画像を 644 × 362 相当で cover → blur σ=64・不透明度 50%。カード上端付近から上へ伸ばし、下方向は透明へフェード
  • カード下のボタン「会員証を変更」までが 1 セット。ボタン・リスト類は本仕様の範囲外

Web プロトタイプとの対応（参考）

  • ジェネレーター（generator/membership/）の HTML/CSS が見た目の正。書き出し PNG と Flutter の描画を見比べてください
  • ぼかしは CSS filter: blur(32px) = Flutter ImageFilter.blur(sigmaX: 32, sigmaY: 32)（stdDeviation とほぼ同義）。上部背景は 64
  • グラデ枠線は CSS では擬似要素 + mask-composite、Flutter では CustomPaint の stroke。どちらも内側 1px
  • 角丸クリップは CSS overflow: hidden + border-radius = Flutter ClipRRect。メイン画像の非対称角丸（上20/下12）に注意
