ブックマークアイコン（動き付き）

記事画面そのものの実装も同梱
・記事画面全体の Flutter コードは未提供です。代わりに 実装指示書（ARTICLE_SCREEN.md） と、 動く HTML リファレンス実装一式（reference-html/） を ZIP に含めています。
・HTML リファレンスはブラウザでも確認できます → 記事プロトタイプを開く

Flutter 組み込みの要点
・Lottie は lottie: ^3.1.2 を追加し、AnimationController の長さを composition.duration に合わせます。
・フレーム f への移動は controller.value = f / 181。ON は 17→90、OFF は 105→181 を animateTo で再生します。
・詳細な手順とサンプルコードは ZIP 内の README.md / ARTICLE_SCREEN.md を参照してください。
