5実装上の注意（落とし穴）

  • 状態はクラス駆動（.s-comment / .s-detail / .vid-landscape）。インラインstyle直接操作はトランジションと競合するので不可
  • ルートは overflow: clip（hidden不可。画面外シートでコンテナがスクロールし bottom:0 が崩れる）
  • 高さは 100dvh 基準（固定844pxではない）
  • ボタンは複製せず1セットを移動（AnimatedPositioned 等）。Lottie二重生成を避ける
  • Lottie ♥🔖 フレーム: LIKE_START=17 / FILLED=90 / UNLIKE_START=105、アニメ中は連打ガード
  • 全画面は Vimeo Player SDK の player.requestFullscreen()（iframeのfullscreenは別挙動）
  • iOS 初回muted対策: 再生タップのジェスチャ内で setMuted(false) → 再生後に再アサート
  • 横展開時、既存の戻るボタンは削除して Vimeo 側に一本化（縦の rail は維持）
縦LIVE の初期状態はコメントシート「閉」（2026-07-04変更）。💬 タップで開閉。アーティスト絞り込みトグルのラベルは 「Artist」。
