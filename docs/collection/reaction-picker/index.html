4アニメーション（この機能の肝）

  • 角丸ボックス自体は下からふわっと: translateY 10px → 0 ／ opacity 0 → 1。transition は opacity 0.28s ease ＋ transform 0.34s cubic-bezier(0.22, 0.61, 0.36, 1)（シート開閉と同じ標準カーブ）
  • 絵文字は左からドミノ式で1つずつ出現: i 番目は 120ms + i×50ms 遅れて開始（ボックスがふわっと出た後に始める）
  • 1絵文字あたり: duration 0.4s、curve cubic-bezier(0.34, 1.56, 0.64, 1)（＝オーバーシュート/バウンド）。opacity 0→1 / scale 0.4→1 / translateY 8px→0
  • 閉じる時: ボックスを逆再生（fade + translateY down）、300ms 後に破棄。ボタンの白枠も同時に解除
  • Flutter: ボックスは AnimationController 1本（slide-up + fade）。絵文字は各 index に Interval をずらした staggered。バウンドは Curves.easeOutBack が cubic-bezier(0.34,1.56,0.64,1) 相当。ディレイ換算 begin = (120 + i×50)/totalMs

6実装上の注意

  • ピッカーは同時に1つだけ。別ボタンを開いたら既存を閉じる。開いたボタンには picking フラグ（＝白枠）を付け、閉じたら必ず外す
  • box-sizing: border-box 前提。白1px枠でボタンの外寸が変わらない（内側に描画）＝レイアウトずれ無し
  • backdrop blur は下に何か描画があって初めて効く。単色ベタ塗りの上では効果が見えない点に注意（実画面はコメント文の上に重なる）
  • リストスクロール開始で即クローズ（追従させない）。コメントシートを閉じる時も一緒に閉じる
  • 絵文字の順序・個数は固定（❤️👍🎉👏✨😆 の6個）。タップされた絵文字を実データのリアクションに加算する連携は別途
