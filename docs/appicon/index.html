BwU App Icon 仕様・素材

Design_System_V2 の BwU ワードマーク（Type=BwU, Color=Dark）をベースに、Apple Human Interface Guidelines に準拠した iOS アプリアイコン一式。背景 #101010、iPhone / iPad / App Store の全サイズと iOS 18 の Dark / Tinted アピアランスを含みます。

01配置仕様（HIG 準拠）

  • キャンバス 1024 × 1024 px・角丸なしフルブリード。角丸マスク（スーパー楕円）は システムが自動適用するため素材側では付けない。
  • ワードマーク幅は キャンバスの 60%（614 px）、天地左右中央配置。横長ロゴの推奨バランスで、左右に 20% ずつのクリアスペースを確保。
  • ロゴの縦寸は約 186 px（18%）。ホーム画面の角丸マスクやフォーカス時の切り欠きでも欠けないセーフゾーン内に収まる。
  • 背景は指定の #101010 単色。グラデーションや写真を使わないため、60 px（Spotlight 40pt 相当）以下でも視認性が落ちない。
  • すべてのサイズは 1024 マスターからの等比スケールで生成（サイズごとのデザイン差し替えなし）。

02ガイドライン準拠チェック

透過なし（App Store 用）1024px マーケティングアイコンはアルファチャンネルなしの不透明 PNG。
角丸を焼き込まないマスクは iOS が適用。素材は正方形フルブリードで書き出し。
シンプルで単一のフォーカスワードマーク 1 要素のみ。小サイズでも判別可能。
テキスト・UI 要素・写真の不使用スクリーンショットやシステム UI 風の表現なし。
iOS 18 アピアランス対応Dark / Tinted 用 1024px を同梱（下記 03 参照）。
全デバイススロット網羅iPhone / iPad / App Store の Xcode 必須サイズをすべて同梱、Contents.json 付き。

03アピアランス（Light / Dark / Tinted）

Light（標準）
#101010 背景 + White ロゴ。デフォルト表示。
Dark
背景透過 + White ロゴ。背景はシステムが暗色グラデーションを供給。
Tinted
グレースケール素材。ユーザーのアクセントカラーでシステムが着色。
